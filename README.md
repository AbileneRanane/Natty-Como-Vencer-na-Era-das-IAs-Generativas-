# Natty-Como-Vencer-na-Era-das-IAs-Generativas-
Parte Escrita de todo o processo de Criação do Projeto "Natty? Como Vencer na Era das IAs Generativas!".



Nome do Projeto:  Data Game: Jogos Divertidos para entender Dados


#Descrição:
-----------

Data Games: Jogos Divertidos para Entender Dados! é um projeto educativo voltado para crianças a partir de 10 anos, com o objetivo de introduzir conceitos básicos de análise de dados de uma maneira divertida e envolvente. Utilizando jogos interativos e atividades lúdicas, o projeto visa ensinar as crianças a coletar, organizar e interpretar dados, transformando informações aparentemente complexas em conhecimentos acessíveis e práticos. Através de uma série de vídeos e materiais didáticos, as crianças aprenderão a criar gráficos, entender padrões e tomar decisões baseadas em dados.
A metodologia do projeto é centrada na gamificação do processo educativo, onde cada módulo de aprendizado é apresentado como um desafio ou missão a ser cumprida. Os jogos são projetados para serem atraentes e estimulantes, mantendo o interesse das crianças enquanto elas exploram o mundo dos dados. Exemplos de atividades incluem a criação de gráficos a partir de pesquisas sobre brinquedos favoritos, análise de dados meteorológicos para planejar atividades ao ar livre e competições de corrida de dados para entender conceitos de velocidade e tempo.




#Tecnologias utilizadas:
-----------------------


-  ChatGPT: Foi usado para criar um tema, titilo e escrever o roteiro.


-  Invideo: Foi utilizado para criar o video inclusos voz e trilha sonora. O prompt Usado foi todo o roteiro que o chatGPT criou.


-  LeonardoIA: Foi utilizado para criar imagens mais exatas como gráficos e listas.


- CapCut: Edição do video.

#Comentário:
------------


Criar um vídeo que pareça natural e legal com a ajuda de um computador inteligente, como uma IA, é um desafio interessante. Essas ferramentas são ótimos para ajudar a fazer coisas rápidas e a encontrar informações, mas fazer um vídeo que pareça feito por pessoas de verdade pode ser difícil.
Para fazer isso, precisamos misturar o que as IAS(Inteligências Artificiais) faz com o que as pessoas fazem melhor. Podemos para escrever partes do vídeo e depois pedir para as pessoas adicionarem suas ideias e emoções. Isso ajuda o vídeo a parecer mais verdadeiro e emocionante, até porque existe imformações exatas que elas as IAS estão aprendendo.
No final, o segredo é trabalhar juntos: pessoas e Inteligência Artificial. Por fim, elas pode dar muitas ideias e ajudar a fazer o vídeo mais rápido, mas as pessoas colocam a criatividade e os sentimentos. Assim, conseguimos fazer um vídeo que é divertido e parece natural.
